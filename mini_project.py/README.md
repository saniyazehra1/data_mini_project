# 🎓 Student Management Mini Project

## 📌 Description
This is a simple Python-based mini project that manages student data.  
It takes input from the user such as name, roll number, and marks, then calculates grades and displays student details.

---

## 🚀 Features
- Add multiple students
- Store student data using list and dictionary
- Automatic grade calculation
- Display all student records
- Show list of passed students (marks ≥ 33)

---

## 🛠️ Technologies Used
- Python
- VS Code

---

## ▶️ How to Run
1. Open the project in VS Code  
2. Run the Python file  
3. Enter number of students  
4. Enter student details (name, roll no, marks)  
5. View output in terminal  

---

## 📊 Grading System
| Marks Range | Grade |
|------------|------|
| > 90       | A    |
| > 80       | B    |
| > 70       | C    |
| > 60       | D    |
| ≥ 33       | E    |
| < 33       | F    |
